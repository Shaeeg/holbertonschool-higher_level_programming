===========================
How to use 1-my_list.py
===========================

Import:
    >>> MyList = __import__('1-my_list').MyList

Normal usage:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

Original list is unchanged:
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Empty list:
    >>> my_list2 = MyList()
    >>> my_list2.print_sorted()
    []

Already sorted:
    >>> my_list3 = MyList([1, 2, 3])
    >>> my_list3.print_sorted()
    [1, 2, 3]

Negative numbers:
    >>> my_list4 = MyList([-3, 0, -1, 2])
    >>> my_list4.print_sorted()
    [-3, -1, 0, 2]
